## 프로젝트 설명 및 내용
 카메라로 마스크를 쓰지 않는 사람들을 **부저와 적색 LED**로 알려주고, 

 **온도가 37.5°C 이상**이 되면 **적색등과 부저**가 울리게 하는 프로젝트입니다.
